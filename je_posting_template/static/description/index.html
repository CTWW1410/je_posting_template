JE Posting Template

Odoo v12


  • This module is a utility to help you create Journal Entries faster based on existing templates



Now, Let's take a look!




  • Well, after you installed the module, an additional menu will appear in the Accounting > Configuration > Posting Template
  • Here, you can declare JE templates. Note that you must confirm the templates before use them.





  • Then, at Journal Entry, you can choose the declared templates for faster input





  • Hope this module is useful to you
